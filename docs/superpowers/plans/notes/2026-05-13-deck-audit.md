# Deck audit for row-hand stacking (2026-05-13)

## Homeworld
- **Ideologies:** heritage, solidarity, sovereignty, transformation (plus wild charters)
- **Land ranks:** 2–9 (all 8 ranks)
- **Role types:** agitator, scholar, preacher, engineer, architect (all 5)
- **Ideology-land combinations:** 4 ideologies × 8 ranks = 32 land cards; each rank available in all 4 ideologies
- **Reachable patterns:** high-card, pair, two-pair, three-of-a-kind, four-of-a-kind, straight, straight (roles), flush, straight-flush, royal-flush
- **Locked out:** full-house (requires 3 of one rank + 2 of a different rank in same row; would need 5 lands minimum in a single row, deck composition allows only 3)
- **Notes:** Complete deck access. All single-ideology pattern types are available. Land-based patterns are constrained only by single-row hand limits (3 lands max per role-filled row).

## Generation Ship
- **Ideologies:** sovereignty, transformation (plus wild charters; no solidarity or heritage)
- **Land ranks:** 2–9 (all 8 ranks)
- **Role types:** agitator, scholar, preacher, engineer, architect (all 5)
- **Ideology-land combinations:** 2 ideologies × 8 ranks = 16 land cards; each rank available in both ideologies
- **Reachable patterns:** high-card, pair, two-pair, straight, straight (roles), flush, straight-flush, royal-flush
- **Locked out:** three-of-a-kind, four-of-a-kind, full-house
- **Notes:** The 2-ideology constraint is intentional per design comment: "2-ideology deck makes Three-of-a-Kind and Four-of-a-Kind mathematically impossible (only 2 Lands per rank exist)." With only 2 ideology options per rank, a player cannot assemble 3+ cards of the same rank (would need 3 copies of the same rank-ideology pair, but only 2 exist). Full-house also becomes unreachable due to this constraint.

## Ruined Homeworld
- **Ideologies:** heritage, solidarity, sovereignty, transformation (plus wild charters)
- **Land ranks:** 2–9 (all 8 ranks)
- **Role types:** agitator, scholar, preacher, engineer, architect (all 5)
- **Ideology-land combinations:** 4 ideologies × 8 ranks = 32 land cards; each rank available in all 4 ideologies
- **Reachable patterns:** high-card, pair, two-pair, three-of-a-kind, four-of-a-kind, straight, straight (roles), flush, straight-flush, royal-flush
- **Locked out:** full-house (same constraint as Homeworld)
- **Notes:** Full deck access identical to Homeworld. Full-house is mechanically impossible: a 3-2 split would require 5 lands in one row, but with 3 role slots (land row + role row + charter row), each row can hold at most 3 cards. To get 3 of rank A + 2 of rank B in a single land row, you'd need 5 land slots in one row, which doesn't exist.

---

## Summary

**Homeworld** and **Ruined Homeworld** unlock 9/10 patterns (all except full-house).
**Generation Ship** unlocks 7/10 patterns due to intentional 2-ideology deck constraint, locking out three-of-a-kind, four-of-a-kind, and full-house.
Full-house is architecturally unreachable across all Settings due to the 3-card-per-row limit: assembling 3+2 requires 5 cards in one row, which is impossible.
